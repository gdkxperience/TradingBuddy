# Trading Buddy - Overview

**Application Name:** Trading Buddy  
**Version:** 1.0.0  
**Type:** Web Application (Single Page Application)  
**Purpose:** Position size calculator for traders to manage risk and calculate optimal trade sizes

## Description

Trading Buddy is a React-based web application that helps traders calculate optimal position sizes based on account size, risk tolerance, and available liquidity. It provides two calculation modes:

1. **Forward Calculation**: Calculate position size starting from account parameters
2. **Reverse Calculation**: Calculate maximum trade size starting from available cash

## Key Capabilities

- Calculate optimal position sizes based on risk tolerance
- Determine maximum trade value from available cash
- Assess trade affordability based on margin requirements
- Visual indicators for safe/risky trades
- Real-time calculations as inputs change

## Target Users

- Day traders
- Swing traders
- Position traders
- Anyone who needs to calculate position sizes for stock trading
